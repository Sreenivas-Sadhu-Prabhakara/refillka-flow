Data intake

Upload pilot details — store signups, waste baselines, partner leads, impact figures, photos, spreadsheets. Submissions are committed to the private refillka-uploads repository. Only the 1–2 people granted write access there can submit; everyone else is refused by GitHub itself.

Getting a token (one minute): GitHub → Settings → Developer settings → Fine-grained personal access tokens → Generate new token → Repository access: only refillka-uploads → Permissions: Contents — Read & write. The token never leaves your browser except to talk to GitHub.
